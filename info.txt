Телеграм бот 'HotelHunter' - @VacationHotelHunterBot по поиску отелей за рубежом.
Основной файл, через который необходимо запустить бота - main.py:
    в нем прописано основное взаимодействие и реакция на действия пользователя с использованием логирования
Описание файлов:
    api.py содержит взаимодействие с API 'booking-com'
    config.py создаёт переменные окружения и включает в себя основные команды бота
    loader.py используется в main.py и принимает на вход токен из config.py
    states.py включает в себя основные состояния
    utils используется в качестве проверки ввода строки
    buttons.py содержит взаимодействие с кнопками в телеграм боте
    .env содержит токен бота и API ключ
